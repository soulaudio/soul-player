# Soul Player Documentation

Welcome to the Soul Player documentation.

## What is Soul Player?

Soul Player is a **local-first, cross-platform music player** built with Rust. It supports:

- Desktop applications (Windows, macOS, Linux)
- Multi-user server streaming
- Embedded hardware (ESP32-S3)

## Architecture

Soul Player uses a trait-based architecture that allows the same core logic to run across all platforms.

```rust
// Simplified example
pub trait AudioOutput {
    fn play(&mut self, buffer: &AudioBuffer) -> Result<()>;
}

// Platform-specific implementations
impl AudioOutput for CpalOutput { ... }
impl AudioOutput for ESP32Output { ... }
```
